Descriptions:

This python script defines 3 functions:

sum_lst calculats sum of all variables in a list.
mean_lst calculats mean of all variables in a list.
dist_lst calculats distance between two points given as lists.

These three functions, together with data imported from 
'2008Male00006.txt'are used in computing George's 
average energy level, location, and total traveled distance 
in his life.

Then it store all data and calculation results in a dictionary.

It also creates a new output file called 'Georges_life.txt'.
A header block, selected contents of data dictionary
(including Date, Time, X and Y coordinates,the Asleep flag,
the behavior mode, and the distance traveled), 
are written into the file in a tab delimited format.
